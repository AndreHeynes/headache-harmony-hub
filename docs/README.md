# Recover & Reclaim Program Documentation

This folder contains comprehensive documentation about the Recover & Reclaim headache management program in markdown format. These documents are designed for internal use, content creation, marketing materials, and video production.

## Available Documents

1. **01-curriculum-overview.md**
   - Complete program structure and phases
   - Daily activities and learning outcomes
   - Key methodologies and approaches
   - Perfect for: Program overviews, curriculum descriptions, onboarding materials

2. **02-phase-one-guide.md**
   - Detailed guide to the first week
   - Assessment questionnaires explained
   - Daily tracking instructions
   - Perfect for: Onboarding videos, introductory content, participant guides

3. **03-program-visualization.md**
   - Patient journey maps
   - Weekly structure breakdowns
   - Assessment timeline
   - Outcome measurement framework
   - Perfect for: Visual content, infographics, flow diagrams

4. **04-research-evidence.md**
   - Scientific foundation and references
   - Validated assessment tools
   - Evidence base for each intervention
   - Complete bibliography
   - Perfect for: Academic presentations, provider materials, credibility content

5. **05-marketing-overview.md**
   - Value propositions and differentiators
   - Target audience profiles
   - Program benefits and outcomes
   - Messaging and positioning
   - Perfect for: Marketing campaigns, sales materials, promotional videos

## Format

All documents are in Markdown (.md) format, which means:
- Easy to copy and paste into any document editor
- Simple formatting that converts well to other formats
- Can be opened in any text editor
- Can be converted to PDF, Word, HTML, etc. using various tools

## Updating Documents

These documents should be updated whenever:
- Program structure changes
- New research evidence becomes available
- Assessment tools are added or modified
- Marketing messaging evolves
- New features are added to the program
